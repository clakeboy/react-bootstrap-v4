# ButtonGroup 组件

ButtonGroup 组件用于将多个按钮组合在一起，使其在视觉上更加紧凑和协调。

## 基本用法

```jsx
import { ButtonGroup, Button } from '@clake/react-bootstrap4';

// 基本按钮组
<ButtonGroup>
    <Button>按钮1</Button>
    <Button>按钮2</Button>
    <Button>按钮3</Button>
</ButtonGroup>
```

## 属性

| 属性名 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| className | string | '' | 自定义类名 |

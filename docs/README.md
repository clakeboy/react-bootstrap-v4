## 关于 React Bootstrap v4

React Bootstrap v4 是一个基于 Bootstrap 4 设计的 React 组件库，提供了现代化、响应式的 UI 组件，用于构建企业级 Web 应用程序。

### 特性

- 基于 React 和 Bootstrap 4
- 丰富的组件集合
- TypeScript 类型支持
- 响应式设计
- 易于定制和扩展

### 支持的组件

本库包含以下主要组件：

- Alerts - 警告组件
- Box - 容器组件
- Button - 按钮组件
- ButtonGroup - 按钮组组件
- Calendar - 日历组件
- CalendarRange - 日期范围选择器
- Card - 卡片组件
- CCheckbox - 复选框组件
- CDropdown - 下拉选择器
- Checkbox - 单选复选框
- CKModal - 模态框组件
- Combo - 组合输入框
- ComboBox - 带搜索的组合框
- Complex - 复杂输入组件
- Container - 容器组件
- DropPanel - 浮动面板
- Dropdown - 下拉菜单
- Form - 表单组件
- GroupStyle - 输入组样式
- HScroll - 水平滚动条
- Icon - 图标组件
- ImageView - 图片查看器
- Input - 输入框组件
- InputGroup - 输入组容器
- InputStyle - 输入样式容器
- Label - 标签组件
- Load - 加载动画组件
- Menu - 菜单组件
- Pagination - 分页组件
- Radio - 单选按钮
- RadioGroup - 单选按钮组
- RImage - 智能图像组件
- Row - 行容器组件
- Scroll - 垂直滚动条
- Select - 下拉选择器
- Switch - 开关组件
- Table - 表格组件
- Tabs - 标签页组件
- TextArea - 文本区域组件
- Title - 标题组件
- Tree - 树形结构组件
